webpack初体验(二)

操作命令

  • npm init
  • npm install webpack@1.12.x --save-dev 安装稳定的指定版本
  • npm install webpack-dev-server --save-dev 安装Webpack 开发工具
  • npm install css-loader style-loader 安装Loader
  • webpack --progress --colors --watch 开启监听模式
  • webpack --display-error-details 打印错误详情

Webpack 中涉及路径配置最好使用绝对路径，建议通过 path.resolve(__dirname, "app/folder") 或 path.join(__dirname, "app", "folder") 的方式来配置，以兼容 Windows 环境。var path = require('path');

项目的目录结构长这样：

  • /app
    main.js
    component.js
  • /build
    bundle.js (自动创建)
    index.html
  • package.json
  • webpack.config.js
