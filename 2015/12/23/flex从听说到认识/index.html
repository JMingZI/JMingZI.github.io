下面开始自己看完后的整理

flex布局，分为基础属性，横向和纵向（x, y）属性,分别说起：

  • 基础属性
    • flex是flex-grow, flex-shrink, flex-basis三个属性的简写，对应的含义分别是：扩展因子、收缩因子、基础大小
    • flex-flow是flex-direction, flex-wrap属性的简写，对应的含义分别是：items的x轴方向，items在y轴上是否换行
    • 他们都是作用在父级元素flex-container上的。
  • x轴上
    • 属性有justify-content，控制items在x轴上的排列方向
  • y轴上
    • 属性是align-items, align-self, align-content，控制items在y轴上的排列方向，其中align-items, align-content是作用在父级元素flex-container上的，align-self是作用在items上的，想想便知道，它是为了覆盖父级元素上的align-items的效果。align-content只有在items有多行的时候有效果，效果和align-items一致。
  • order属性
    • 用整数值来定义排列顺序，数值小的排在前面。可以为负值。默认是0

关于扩展因子flex-grow的计算方法：
扩展因子的意思就是，把多余的空间再按照因子分配，例如box宽度400px，下面三个div分别为100px，那么多余100px，此时flex-grow就生效了，而flex-shrink没效。如果三个div的flex-grow比例是1：1：3，那么多余的分配就为100(1/5)、100\(1/5)、100*(3/5)。 然后加上 flex-basis

这个还比较好理解。不好理解的就是收缩因子的计算。

flex-shrink在元素items溢出的情况下分配空间。如果flex-containerbox的宽度是400px，下面3个div宽度分别是200px，那么超出了父元素宽度的200px，此时怎么计算？
假设比例是1：1：3，那么超出的总宽度为2001+200\1+200*3 = 1000px
每个div需要移除的px为，2001/1000 \ 200 = 40px, 200*1/1000 200 = 40px, 200\3/1000 * 200 = 120px,那么实际的宽度就为160px,160px,80px。

另外

  • 如果你想让元素仅仅使用它本身的宽度，比如按钮，则设置 flex: none，none 将会被解释为 0 0 auto.
  • 如果想要一个固定大小的元素，则设置 flex: 0 0 size。如：flex 0 0 60px。
  • 如果你想让元素自动扩展到可以利用的空间，如果有多个这种类型的元素，它们可以平均分配空间，则设置 flex: auto，它代表 1 1 auto.
